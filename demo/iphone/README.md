## Features

- **Camera mode**: AR-based 3D detection with LiDAR depth alignment
- **Upload mode**: Detect objects in photos from your library
- **RGB / RGBD modes**: Monocular or depth-enhanced detection
- **Interactive boxes**: Tap to select, view dimensions, delete individual detections
- **Open-vocabulary**: Detect any object category via text prompts
